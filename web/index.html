🚀 GitHub 文件加速

直接替换 GitHub 域名即可加速下载文件、克隆仓库等操作

方式一：替换域名

https://github.com/user/repo/file → https://your-domain.com/github.com/user/repo/file

方式二：使用路径前缀

https://github.com/user/repo/file → https://your-domain.com/github/user/repo/file

Git Clone 加速

git clone https://your-domain.com/github.com/user/repo.git

💡 小提示

  • 支持自动转换 blob 链接为 raw 下载链接
  • 支持 Release 文件下载加速
  • 支持 API 访问加速
  • 文件大小限制：- MB

🐳 Docker 镜像加速

支持多个镜像仓库的加速代理服务

Docker Hub (默认)

docker pull your-domain.com/nginx:latest

GitHub Container Registry

docker pull your-domain.com/ghcr.io/user/image:tag

Google Container Registry

docker pull your-domain.com/gcr.io/project/image:tag

Kubernetes Registry

docker pull your-domain.com/registry.k8s.io/image:tag

💡 支持的镜像仓库

  • ✅ Docker Hub (docker.io) - 默认
  • ✅ GitHub Container Registry (ghcr.io)
  • ✅ Google Container Registry (gcr.io)
  • ✅ Kubernetes Registry (registry.k8s.io)
  • ✅ NVIDIA Container Registry (nvcr.io)
  • ✅ Quay.io (quay.io)
  • ✅ Microsoft Container Registry (mcr.microsoft.com)
  • ✅ Elastic Container Registry (docker.elastic.co)
